Інструменти які я використовую
PhpStorm
Мій самий головний інструмент, навіть не уявляю без нього роботи. Надзвичайно зручно писати код, підсвітка синтаксису, легке переключення між файлами і робота з ними. Зручність не тільки в роботі з PHP, але і в верстці сайтів. Деколи таке відчуття що він замість мене пише код.
Brackets
Самий перший редактор коду. Завдяки плагінам і підсвітці синтаксису я досі ним користуюсь як допоміжним. Використовував чисто для верстки сайтів до того як почав вивчати backend розробку.
Open Server Panel
Локальний сервер для роботи з PHP і базами данних. Дуже зручна програма для віндовс. Хороший і зручний інтерфейс.
MySQL
Це система для управління базами данних. Використовую з перших днів. Таксамо зручний і зрозумілий інтерфейс.
Avocode
Це сервіс для роботи з дизайном. Коли діло доходить до роботи з дизайном то краще нього я не знайшов. Таксамо використовую з перших днів.
